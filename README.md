# MediHistory

MediHistory is a starter healthcare management platform with a React frontend and Spring Boot backend. The app includes patient dashboards, report upload, appointment booking, and emergency QR-style health data.

## Run the project

### Backend
```bash
cd backend
mvn spring-boot:run
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Stack
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Spring Boot + Java
- Data: Mock endpoints for quick local development
- Deployment: Ready for Docker / AWS EC2
